In this Data Analysis Task, we worked with a retail dataset to answer key business questions using both SQL queries and Excel visualizations. We began by preparing the dataset and calculating the total revenue generated from all transactions, which provided a baseline measure of overall sales performance. From there, we identified the top-selling product based on total units sold and determined which product generated the highest revenue, highlighting the difference between high-volume items and high-value items. We then analyzed regional performance to find the best-performing region by revenue and extended the analysis to identify the top-performing salesperson, showing who contributed most to overall success. Beyond these direct queries, we examined trends over time to see how sales evolved across different months and regions, uncovering seasonal patterns and shifts in customer demand. From this analysis, we provided business insights such as which product categories consistently drive revenue and which regions show growth potential. Finally, we created charts to visually represent sales performance, making the insights easier to interpret and communicate. Altogether, this project demonstrated how raw transaction data can be transformed into actionable insights by combining SQL analysis with Excel visualization, building a structured framework that highlights product performance, regional strengths, and sales trends for data-driven decision-making.
